TSB Branch location data

Hello,

I've noticed that TSB use OpenStreetMap on your web site to show where your branches are located, and I think this is great. However, while TSB uses its own gps data to pinpoint each store, on the actual map, not all locations are marked up correctly.

I am writing to you in the hope that you will help assist with the OpenStreetMap project, by providing an CSV File (Comma Separated Values File [.csv]) of all your branches you currently have within the United Kingdom.

OpenStreetMap, as you know, is similar to Google Maps or Apple Maps, and aims to maintain an accurate map of the world, with accurate data points for things like banks, shops, restaurants, etc. The difference is that OpenStreetMap is an "open" platform that allows anybody to edit the map (similar to Wikipedia for articles).

At the moment, OpenStreetMap (like other maps) has listings for TSB that are either incorrect, out of date, or missing. Such things might include the wrong opening hours, the wrong address data, or the fact that a branch has since closed. I've included two screen shots of where this is shown, one being a branch that was still marked up as "Cheltenham & Gloucester".


This page (https://osm.mathmos.net/ghosts/lloyds-tsb/) also highlights many entries that still list "Lloyds TSB" as a single bank.

This past data will have been obtained from local people visiting the location and noting the location, address, and office hours which are usually attached to the branch.

To help maintain accurate branch location data for TSB on OpenStreetMap, I would like to ask you to consider, on a regular (live / weekly / monthly) basis, to release a full list of your current branch locations and office hours, in a CSV format, so that editors of OpenStreetMap are able to check against a known up to date list of your branches to ensure they remain accurate.

By making available a regular up to date list in a computerised format, OpenStreetMap editors can quickly and easily see if branch data has changed over time (such as different office hours) and update the map accordingly.

If possible, under your "branch location" page, placing a link on this page to a "download" link for the CSV file would be the simplest way of making this list available, with a note of when the file was last updated.

This link would also need to be accompanied by a paragraph or two of text granting permission for the branch location data to be used by others for the purpose of maintaining accurate branch listings on third party platforms, meaning that people who wish to use it to update OpenStreetMap (or other listings) know that you have allowed this to occur.

The CSV list would need to contain the following values, one per line for each branch:

BRANCH_CODE_ID,
BRANCH_NAME,
BRANCH_TYPE,
BRANCH_ADDRESS_LINE_1,
BRANCH_ADDRESS_LINE_2,
BRANCH_ADDRESS_LINE_3,
BRANCH_ADDRESS_LINE_TOWN-CITY,
BRANCH_ADDRESS_LINE_COUNTY,
BRANCH_ADDRESS_POSTCODE,
BRANCH_LONGITUDE,
BRANCH_LATITUDE,
MONDAY_OPEN,
MONDAY_CLOSE,
TUESDAY_OPEN,
TUESDAY_CLOSE,
WEDNESDAY_OPEN,
WEDNESDAY_CLOSE,
THURSDAY_OPEN,
THURSDAY_CLOSE,
FRIDAY_OPEN,
FRIDAY_CLOSE,
SATURDAY_OPEN,
SATURDAY_CLOSE,
SUNDAY_OPEN,
SUNDAY_CLOSE

... it could also include other useful information that you feel is relevant and helpful, such as:

BRANCH_BRAND,
BRANCH_ATM_INSIDE,
BRANCH_ATM_OUTSIDE,
BRANCH_PHONE,
BRANCH_EMAIL,
BRANCH_WEBSITE,
BRANCH_MANAGER,
BRANCH_WIFI,
BRANCH_LUNCH_BREAK,
BRANCH_OPENED_DATE,
BRANCH_CLOSING_DATE,

... where opened date was when the branch first opened, and closing date might be if the branch is set to close down.

If you are willing to participate in the regular release of up to date branch information, I feel it would greatly help with the accurate and up to date mapping of the United Kingdom by the many committed OpenStreetMap editors.

Thank you for at least taking the time to read this email.
